How to Play

Objective:

The aim of Color Code Challenge is to match a randomly generated RGB color code to its corresponding color among a set of options. You will be presented with a color code in the format RGB(r, g, b) and your task is to identify the color that matches that code.

RGB Color Code:

RGB stands for Red, Green, Blue - the primary colors used in digital displays. Each color channel is represented by a number from 0 to 255. For example, RGB(12, 167, 200) indicates a color composed of 12 units of red, 167 units of green, and 200 units of blue.

How to Identify Colors:

To play the game, you need to be able to recognize colors based on their RGB components. Here's a simple guide to get you started:

Red (R): Higher R value with lower G and B values will create shades of red.

Green (G):Higher G value with lower R and B values will create shades of green.

Blue (B): Higher B value with lower R and G values will create shades of blue.

Mixing Colors: Combining different R, G, and B values will result in a wide range of colors.

Choosing Between HEX and RGB:

You have the option to display color codes in either HEX or RGB format. HEX codes are alphanumeric codes representing colors, while RGB uses numeric values. Select the format that you are more comfortable with.

Adjusting Difficulty:

The game lets you choose the number of colors displayed as options. Higher color counts increase the difficulty. Start with a lower number of colors and gradually increase the challenge as you improve your color-matching skills.

How to Play (Step-by-Step):

1. Read the RGB color code displayed on the screen (e.g., RGB(12, 167, 200)).

2. Observe the set of colors shown below the code.

3. Identify the color that matches the given RGB code.

4. Select the correct color by clicking on it.

5. If your choice is correct, you advance to the next round. Otherwise, try again until you succeed.

Remember: Practice makes perfect! Keep playing to hone your color recognition abilities and become a master of the Color Code Challenge.

نحوه بازی: چالش کد رنگ

هدف:

هدف از بازی چالش کد رنگ، اتصال یک کد رنگ RGB تصادفی به رنگ مربوطه‌اش در میان گزینه‌های داده‌شده است. یک کد رنگ به شما ارائه می‌شود که به صورت RGB(r, g, b) نمایش داده می‌شود و وظیفه شما شناسایی رنگی است که با آن کد تطابق دارد.

کد رنگ RGB:

RGB مخفف Red، Green، Blue می‌باشد که رنگ‌های اولیه استفاده شده در نمایشگرهای دیجیتال هستند. هر کانال رنگ به عددی از 0 تا 255 نمایش داده می‌شود. به عنوان مثال، RGB(12, 167, 200) رنگی را نمایش می‌دهد که از 12 واحد قرمز، 167 واحد سبز و 200 واحد آبی تشکیل شده است.

شناسایی رنگ‌ها:

برای بازی کردن، شما باید بتوانید رنگ‌ها را بر اساس اجزای RGB آن‌ها شناسایی کنید. اینجا یک راهنمای ساده برای شروع بازی آمده است:

قرمز (R): مقدار بالاتر R با مقادیر پایینتر G و B، نقاط قرمزی ایجاد می‌کند.

سبز (G): مقدار بالاتر G با مقادیر پایینتر R و B، نقاط سبزی ایجاد می‌کند.

آبی (B): مقدار بالاتر B با مقادیر پایینتر R و G، نقاط آبی ایجاد می‌کند.

ترکیب رنگ‌ها: ترکیب مقادیر مختلف R، G و B، باعث ایجاد طیف گسترده‌ای از رنگ‌ها می‌شود.

انتخاب بین کد HEX و RGB:

شما دارای گزینه‌ای هستید تا کدهای رنگ را به صورت HEX یا RGB نمایش دهید. کدهای HEX کدهای الفبایی عددی هستند که رنگ‌ها را نمایش می‌دهند، در حالی که RGB از مقادیر عددی استفاده می‌کند. فرمتی را انتخاب کنید که با آن آشنایی دارید.

تنظیم سختی:

این بازی به شما اجازه می‌دهد تعداد رنگ‌های نمایش داده‌شده به عنوان گزینه‌ها را انتخاب کنید. تعداد رنگ‌های بیشتر موجب افزایش سختی می‌شود. با تعداد کمتری رنگ شروع کنید و با بهبود مهارت‌های تطبیق رنگ‌ها، تدریجی سختی را افزایش دهید.

نحوه بازی (مراحله به مرحله):

    1. کد رنگ RGB نمایش‌داده‌شده را روی صفحه بخوانید (به عنوان مثال، RGB(12، 167، 200)).

    2. مجموعه‌ای از رنگ‌ها را که زیر کد نمایش داده‌شده است، مشاهده کنید.

    3. رنگی را که با کد RGB داده‌شده تطابق دارد، شناسایی کنید.

    4. رنگ صحیح را با کلیک بر روی آن انتخاب کنید.

    5. اگر انتخاب شما صحیح باشد، به مرحله بعدی پیش خواهید رفت. در غیر اینصورت، تا موفقیت ادامه دهید.

یادآوری: تمرین باعث پیشرفت می‌شود! به بازی کردن ادامه دهید تا توانایی‌های تشخیص رنگ خود را بهبود بخشید و به استاد چالش کد رنگ تبدیل شوید
